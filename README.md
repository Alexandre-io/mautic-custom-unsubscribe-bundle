# Mautic Custom Unsubscribe for Mautic 

## Installation

### Manual

1. Use last version
2. Unzip files to plugins/MauticCustomUnsubscribeBundle
3. Clear cache (app/cache/prod/)
4. Go to /s/plugins/reload
